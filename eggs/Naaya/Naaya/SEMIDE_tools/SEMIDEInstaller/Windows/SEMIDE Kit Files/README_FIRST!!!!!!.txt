CUM se face kitul de SEMIDE:

1. se copiaza cu totul folderul SEMIDE_KIT_FILES in alt folder
2. se sterg toate fisierele svn
3. se copiaza in /instance/Products toate produsele necesare SEMIDE (se suprascrie tot)
4. se pune peste in Products ce se gaseste in folderul original SEMIDE_KIT_FILES/instance/Products:
	sunt fisiere personalizate pt SEMIDE toolkit
5. se sterg toti portletii legati de Country din SEMIDE/skel/portlets
6. se sterge NyCountry din NaayaContent

this is it!!!
