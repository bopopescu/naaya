This folder contain the source code for

1. dnt_rgis.swf 
	Destinet GIS viewer (used to display and query Destinet sites onto the map)

2. dntp_rgis.swf 
	Destinet GIS picker (used to add by coordinate picking Destinet sites on the map)

Both application are written using Macromedia Flash 8.

The compiled flash movies are located into /www folder
