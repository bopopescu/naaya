Usage:

- download ZopeTestCase from http://www.zope.org/Members/shh/ZopeTestCase;
- Extract the tarball into the lib/python/Testing directory of your Zope installation;

Windows
=======
- set up the environment variables in 'run_tests.bat' to reflect your system;
- run 'run_tests.bat' and see the results.

UNIX
====
- run './runtest runalltests.py' and see the results.
