Once installed the DOMSeeker adds itself to the right click menu of Internet Explorer

To install it:
	- if you put all the files directly in c:\ you should just execute the z.reg file and restart the Internet Explorer
	- if not:
		- inside the z.reg file set the path to the z.html file
		- inside the z.html file set the path to z.js file
		- execute the z.reg file
		- restart the Internet Explorer
